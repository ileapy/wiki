# 消费接口

消费是指境内外持卡人在境内外商户网站进行购物等消费时用银行卡结算的交易，经批准的消费额将即时 地反映到该持卡人的账户余额上。

> 阅读本文之前请仔细阅读[手机支付控件 - 消费接口](https://open.unionpay.com/tjweb/acproduct/APIList?apiservId=450&acpAPIId=765&bussType=0)

## 接口调用

返回结果自行验签

```php
$params = [
    'orderId' => date('YmdHis'), // 订单号
    'txnAmt' => 1, // 支付金额 单位为分
];

$res = $app->order->pay($params);

// 验签
if ($app->signature->validate($res)) {
    echo "验签成功";
}else
{
    echo "验签失败";
}

```
